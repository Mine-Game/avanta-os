# Avanta OS (Scaffold v0)

Каркас интерфейса операционной системы из 3 модулей:
- **Ops** (операционный модуль)
- **Brain** (мозговой модуль)
- **Lab** (лабораторный модуль)

## Что уже реализовано

- Визуальный стиль под **macOS-like desktop**
- Плавающий **dock** внизу с крупными иконками
- Для каждого модуля — стартовая **dashboard-страница** с функциями
- Вкладки внутри модуля:
  - нельзя pin/unpin
  - можно **перетаскивать и менять порядок**
- В Ops добавлена первая рабочая область: **Mission Control**
  - показывает модель
  - показывает активные сессии
  - показывает обзор cron-задач и их статус
- Кнопка обновления статуса работает только с live-источником `/api/mission-control`
- Источник live-данных реализован в `server.js` (читает `openclaw status/sessions/cron` в реальном времени)
- Принцип проекта: **никаких ложных/mock-данных в интерфейсе**; при отсутствии источника показывается явный статус `source not connected`

## Запуск

Откройте `index.html` в браузере.

Или через локальный сервер (рекомендуется).

### Фиксированный preview-порт Avanta-OS: **8800**

Для Avanta-OS используем только адрес: **http://127.0.0.1:8800/** (не менять):

```powershell
cd projects/avanta-os
./start-preview.ps1
```

Фоновый запуск (если хочешь просто «убедиться, что поднято»):

```powershell
cd projects/avanta-os
./start-preview-bg.ps1
```

После запуска открывай: <http://127.0.0.1:8800/>

> Технический маркер порта хранится в файле `PREVIEW_PORT`.
